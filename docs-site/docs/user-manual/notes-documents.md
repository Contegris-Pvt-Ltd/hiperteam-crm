# Notes & Documents

**Notes** and **Documents** are available on virtually every record type in HiperTeam CRM. They provide a place to capture free-text observations and attach files directly to the records they relate to.

## Notes Panel

The Notes panel appears as a tab on detail pages for contacts, leads, opportunities, accounts, tasks, and projects.

### Creating a Note

1. Open any record detail page.
2. Go to the **Notes** tab.
3. Click **+ Add Note**.
4. Type your note content in the text area.
5. Click **Save**.

![Screenshot: Notes tab on a lead detail page with a text input area and saved notes below](../../static/img/screenshots/shared/notes-panel.png)

### Viewing Notes
Notes are displayed in **reverse chronological order** (newest first). Each note shows:
- The **content** of the note
- **Author** — who wrote it (with avatar)
- **Timestamp** — when it was created
- **Edit/Delete** actions (if you are the author or an admin)

### Editing Notes
1. Click the **edit icon** on your note.
2. Modify the content.
3. Click **Save**.

:::note
You can only edit notes you created, unless you have administrator permissions. Edited notes show an "edited" indicator.
:::

### Deleting Notes
1. Click the **delete icon** on your note.
2. Confirm the deletion.

:::tip
Use notes to record meeting summaries, phone call outcomes, important decisions, and action items. Notes on records create a permanent, searchable record of your interactions that your entire team can reference.
:::

### Best Practices for Notes
- **Be specific** — "Called John, discussed Q2 renewal. He is evaluating competitor X. Follow-up scheduled for Friday." is much more useful than "Called John."
- **Tag action items** — mention follow-ups explicitly so they are easy to find
- **Note context** — explain the "why" behind decisions and observations

## Documents Panel

The Documents panel lets you upload and manage files attached to CRM records.

### Uploading Documents

1. Open any record detail page.
2. Go to the **Documents** tab.
3. Click **Upload** or drag and drop files onto the upload area.
4. Files upload and appear in the document list.

![Screenshot: Documents tab showing uploaded files with names, sizes, and download buttons](../../static/img/screenshots/shared/documents-panel.png)

### Downloading Documents
Click the **file name** or the **download icon** next to any document to download it to your computer.

### Viewing Document Details
Each document entry shows:
- **File name**
- **File size**
- **File type** icon
- **Upload date**
- **Uploaded by** — the user who uploaded it

## Supported File Types

HiperTeam CRM supports a wide range of file types:

| Category | Formats |
|---|---|
| **Documents** | PDF, DOC, DOCX, TXT, RTF |
| **Spreadsheets** | XLS, XLSX, CSV |
| **Presentations** | PPT, PPTX |
| **Images** | JPG, JPEG, PNG, GIF, SVG |
| **Archives** | ZIP, RAR |
| **Other** | Any common file format |

:::warning
There may be file size limits configured by your administrator. Very large files (above the limit) will be rejected during upload. If you need to share large files, consider using a file sharing service and linking to it in a note.
:::

## Document Organization

Documents on a record are listed chronologically. To find a specific document:

- **Scroll** through the list
- **Search** using the document search field (if available)
- Look at **file type icons** for visual identification

:::info
Documents attached to a record are accessible to anyone who has permission to view that record. Be mindful about uploading sensitive documents — they will be visible to all users with access to the parent record.
:::
