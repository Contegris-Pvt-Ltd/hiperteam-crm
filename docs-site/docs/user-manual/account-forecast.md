# Account Forecast Report

The Account Forecast provides a comprehensive view of your sales pipeline grouped by account, with forecast category breakdowns for revenue planning.

Access it from **Reports → Account Forecast** in the sidebar.

![Screenshot: Account Forecast report](../../static/img/screenshots/user/account-forecast.png)

## Controls

### Time Period
Toggle between:
- **Current Quarter** — Active quarter pipeline
- **Next Quarter** — Upcoming quarter outlook

### Scope (Managers & Admins)
- **My Deals** — Only your opportunities
- **My Team** — Your team's opportunities
- **All** — Organization-wide

### Search
Filter by account name or industry.

## Summary KPIs

| KPI | Description |
|---|---|
| **Total Accounts** | Number of accounts with active deals |
| **Open Deals** | Count of opportunities in the pipeline |
| **Total Pipeline** | Sum of all deal amounts |
| **Weighted Value** | Pipeline adjusted by probability |
| **Avg Probability** | Average close probability across deals |

## Forecast Categories

Deals are categorized by likelihood:

| Category | Description | Color |
|---|---|---|
| **Commit** | High confidence — verbal/written commitment | Green |
| **Best Case** | Strong pipeline — likely to close | Blue |
| **Pipeline** | Active opportunities — standard probability | Amber |
| **Upside** | Early stage — potential but uncertain | Purple |

A **stacked bar** shows the distribution of pipeline value across categories.

## Three Views

### By Account
Expandable rows showing each account with:
- Total amount, deal count, weighted value
- Category breakdown table when expanded
- Click to navigate to account detail

### By Forecast Category
Expandable rows grouped by category:
- Total deals, amount, and weighted value per category
- Account list when expanded

### Flat Table
Sortable table with one row per account:
- Account, Industry, Deal Count, Total Amount, Weighted Value
- Per-category amounts (Commit, Best Case, Pipeline)
- Average Probability, Owner
- Click column headers to sort
