# Mobile App (PWA)

IntelliSales CRM is a **Progressive Web App (PWA)**, which means you can install it on your phone or tablet directly from your browser — no app store required. Once installed, it looks and behaves like a native mobile app with its own icon, splash screen, and mobile-optimized navigation.

## What is a PWA?

A Progressive Web App runs in your browser but can be "installed" to your device's home screen. Benefits include:

- **No app store** — install directly from the browser
- **Always up to date** — uses the latest version automatically
- **Works offline** — basic navigation and cached data available without connectivity
- **Native feel** — full-screen mode, home screen icon, push notifications

## Installing on Android

1. Open IntelliSales CRM in **Chrome** on your Android device.
2. You may see a banner at the bottom saying **"Add IntelliSales CRM to Home screen"** — tap **Install**.
3. If no banner appears:
   - Tap the **three-dot menu** (top right) in Chrome.
   - Select **"Add to Home screen"** or **"Install app"**.
4. Confirm the installation.
5. The IntelliSales CRM icon appears on your home screen.
6. Tap the icon to launch the app in full-screen mode.

![Screenshot: Android Chrome install prompt for IntelliSales CRM](../../static/img/screenshots/mobile/android-install.png)

## Installing on iOS (iPhone / iPad)

1. Open IntelliSales CRM in **Safari** on your iOS device.
2. Tap the **Share** button (square with upward arrow) at the bottom of the screen.
3. Scroll down and tap **"Add to Home Screen"**.
4. Optionally edit the name, then tap **Add**.
5. The IntelliSales CRM icon appears on your home screen.
6. Tap the icon to launch the app in full-screen mode.

![Screenshot: iOS Safari share sheet with Add to Home Screen option](../../static/img/screenshots/mobile/ios-install.png)

:::note
On iOS, you must use **Safari** to install PWAs. Chrome, Firefox, and other browsers on iOS do not support the "Add to Home Screen" feature with full PWA capabilities.
:::

## Mobile Navigation

On mobile devices, IntelliSales CRM uses a **bottom navigation bar** for quick access to key modules:

| Icon | Module |
|---|---|
| Home | Dashboard |
| Users | Contacts |
| Building | Accounts |
| Target | Leads |
| Briefcase | Opportunities |
| More | Menu for all other modules |

The bottom navigation replaces the desktop sidebar, providing thumb-friendly access to the most-used areas.

![Screenshot: Mobile view showing bottom navigation bar with module icons](../../static/img/screenshots/mobile/mobile-bottom-nav.png)

## Mobile-Responsive Features

The entire IntelliSales CRM interface is responsive. Key adaptations on mobile include:

- **Data tables** scroll horizontally with pinned key columns
- **Detail pages** stack sidebar and content vertically
- **Forms** use full-width fields optimized for touch input
- **Modals** expand to full screen on small devices
- **Charts and dashboards** reflow to single-column layout

:::tip
For the best mobile experience, use the app in portrait mode. Landscape mode works but some tables and forms are easier to navigate in portrait.
:::
